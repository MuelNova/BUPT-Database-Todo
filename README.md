# BUPT-Database-Todo
应付 21-BUPT-Database 选作大实验的作品



利用 MongoDB 实现了 Todo 的后端，利用 beanie + FASTAPI 实现异步操作。

## Features

- [X] Todo 的增删查改
- [X] Todo 的共享
- [X] JWT 支持的 User 的创建与登录
- [x] FASTAPI 框架
- [x] beanie 异步数据库驱动



## Run

```bash
git clone https://github.com/MuelNova/BUPT-Database-Todo.git
cd BUPT-Database-Todo
pdm install
pdm run main.py
```
